GTUI

Gmail TUI client — 9 Catppuccin Mocha variants

GitHub Install

Flavours

Features

  • ✓ Gmail API (read, send, modify)
  • ✓ OAuth 2.0 authentication
  • ✓ 9 Catppuccin Mocha accent variants
  • ✓ Responsive icon-only mode
  • ✓ Nerd Font icons
  • ✓ Folder navigation
  • ✓ Compose & send emails
  • ✓ Mouse support (Textual)

Install

git clone https://github.com/OfficalConus/gtui.git
cd gtui
python3 gtui.py

Requires OAuth credentials in ~/.gmail-tui/client_secret.json
